Course Objectives: 
1. To give a comprehensive understanding of the compiler construction process, including lexical 
analysis, parsing, semantic analysis, code generation, and optimization. 
2. To enable the learner to design, implement, and test a compiler for a simple language, utilizing 
tools including LEX and YACC for lexical analysis and parsing, respectively. 
3. To learn to use code optimization techniques to improve the efficiency of generated code.
