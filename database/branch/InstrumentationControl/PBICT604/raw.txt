Course Objectives: 
1. To equip students with advanced techniques in designing cascade compensators using root-
locus and Bode plot methods. 
2. To enable students to model and analyse continuous linear time-invariant (LTI) systems using 
state-space representations

Hours 
1 
Design of Cascade Compensators (10 Hours) 
Dominant pole; realisation of basic compensators: lead, lag, lag, and lag-
lead compensator. 
Cascade compensation using root-locus: lead, lag, and lead-lag 
compensation. 
Cascade compensation using bode plot: lag, lead, and lag-lead compensation. 
 
10 
2 
State Space Modelling of Continuous Linear Time-Invariant Systems 
(9 Hours) 
Concept of state, state variable, state vector and state space; State models 
for linear continuous-time systems; State variable representation of 
continuous-time systems using physical variables. 
Transfer function from state variable model; transformation of state 
variables and invariance property. 
State space representation using phase variable (state space 
representation from transfer function): Controllable canonical form, 
observable canonical form, Jordan Canonical Form. 
 
9 

 
 
3 
State Variable Analysis of Continuous LTI Systems (8 Hours) 
Solution of state equation; state transition matrix; properties of state 
transition matrix. 
Computation of the state transition matrix using Lyapunov’s 
Method and Cayley Hamilton theorem; Response of homogeneous and 
non-homogeneous state-space model. 
Concept of controllability and observability; Kalaman test. 
8 
4 
State Variable Design (Continuous) (6 Hours) 
Design of state variable feedback if the given model is in the controllable 
canonical form; design of arbitrary state variable feedback if the given 
model is not in controllable canonical form. 
Design of full order state observer; application of duality Principle; 
concept of reduced order state-observer. 
Stability analysis of Continuous LTI Systems using Phase Plane 
Method (3 Hours) 
Singular points, classification of singular points. 
 
9
