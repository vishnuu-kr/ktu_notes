MULTISTAGE AND DIFFERENTIAL AMPLIFIERS: 
Multi stage amplifier - Differential amplifier- basic MOSFET differential 
pair operation. (3 Hours) 
FEEDBACK AMPLIFIERS: Effect of negative feedback on bandwidth, 
gain and noise reduction. Voltage, Current, Transconductance, and 
Transresistance amplifiers. (5 Hours) 
POWER AMPLIFIERS: classifications (basic operation using input output 
signal and circuit diagram) and power efficiency - class A (series fed/ 
transformer coupled) - class B (transformer coupled push pull amplifier, 
complementary symmetry) - class C - power amplifier (3 Hours)
