Course Objectives: 
1. The skill to write Python Programs in an efficient way.  
2. The knowledge to use Python for statistical computation, modelling, analysis and graphics. 
3. The ability to implement multivariate statistical analysis techniques using Python.

 Linux

PRACTICE QUESTIONS  
1. Ask the user for a number. Depending on whether the number is even or odd, print out an 
appropriate message to the user. 
2. Create a program that asks the user for a number and then prints out a list of all the divisors of 
that number. 
3. Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string 
is valid. An input string is valid if: a. Open brackets must be closed by the same type of 
brackets., b. Open brackets must be closed in the correct order. c. Every close bracket has a 
corresponding open bracket of the same type. 
4. Create a Python dictionary representing a student's grades in different subjects (e.g., Math, 
Science, History). Write a function to calculate the average grade of the student across all 
subjects. 
5. Define a tuple containing the names of different fruits. Write a Python program that prompts 
the user to enter a fruit name. If the entered fruit exists in the tuple, display a message 
confirming its presence; otherwise, display a message indicating its absence. 
6. Count the number of vowels, consonants, words and question marks in a given string. 
7. Read a number in decimal and convert it into different bases. 
8. Print the multiplication table of a given number. ( use while loop) 
9. Find nth Fibonacci number using recursion. 
10. Read and display a sparse matrix using dictionary 
11. Pivot Tables and Cross Tabulations 
12. The areas of the various continents of the world (in millions of square miles) are as follows:11.7 
for Africa; 10.4 for Asia; 1.9 for Europe; 9.4 for North America; 3.3 Oceania; 6.9 South 
America; 7.9 Soviet Union. Draw a bar chart representing the given data. 
13. Plot a graph y = f(x). 
14. How to get all 2D diagonals of a 3D NumPy array? 
15. Flatten a 2d numpy array into 1d array 
16. NumPy – Fibonacci Series using Binet Formula 
17. Inverse a matrix using NumPy 
18. Calculate inner, outer, and cross products of matrices and vectors using NumPy 
19. Compute the Kronecker product of two multidimensional NumPy arrays 
20. Convert the matrix into a list 
21. Calculate the QR decomposition of a given matrix using NumPy 
22. Compute the eigenvalues and right eigenvectors of a given square array using NumPy? 
23. Calculate the n-th order discrete difference along the given axis 

 
 
24. Evaluate Einstein’s summation convention of two multidimensional NumPy arrays 
25. Compute pearson product-moment correlation coefficients of two given NumPy arrays 
26. Calculate the average, variance and standard deviation in Python using NumPy 
27. Plot line graph from NumPy array 
28. Convert a NumPy array into a csv file 
29. Write a program to count the number of words, sentences, upper case letters, lowercase letters 
and special symbols in a text stored in file.

Continuous Assessment (25 Marks) 
1. Preparation and Pre-Lab Work (7 Marks) 
 
Pre-Lab Assignments: Assessment of pre-lab assignments or quizzes that test 
understanding of the upcoming experiment. 
 
Understanding of Theory: Evaluation based on students’ preparation and understanding 
of the theoretical background related to the experiments. 
2. Conduct of Experiments (7 Marks) 
 
Procedure and Execution: Adherence to correct procedures, accurate execution of 
experiments, and following safety protocols. 
 
Skill Proficiency: Proficiency in handling equipment, accuracy in observations, and 
troubleshooting skills during the experiments. 
 Teamwork: Collaboration and participation in group experiments. 
3. Lab Reports and Record Keeping (6 Marks) 
 
Quality of Reports: Clarity, completeness and accuracy of lab reports. Proper 
documentation of experiments, data analysis and conclusions. 
 
Timely Submission: Adhering to deadlines for submitting lab reports/rough record and 
maintaining a well-organized fair record. 
 
4. Viva Voce (5 Marks) 
 Oral Examination: Ability to explain the experiment, results and underlying 
principles during a viva voce session. 
Final Marks Averaging: The final marks for preparation, conduct of experiments, viva, and 
record are the average of all the specified experiments in the syllabus.
