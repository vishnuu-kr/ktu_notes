Course Objectives: 
1. Provide an overview of cloud robotics 
2. Understand the concept of   programmable robots for automated tasks, principles of cloud 
computing technologies and robotics-related software paradigm such as the Robotics 
Operating System (ROS) 
3. Provide knowledge on how cloud computing techniques can automate manufacturing tasks 
using algorithms designed based on machine learning and big data analytics.
