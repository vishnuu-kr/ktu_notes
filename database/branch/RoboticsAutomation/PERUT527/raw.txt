Course Objectives: 
1. To familiarise with the measurement system and mechanism for measuring parameters

       Programming in C Course Type

Course Objectives: 
1. Understand the principles of algorithm analysis, including time and space complexity. 
2. Analyse the complexity of algorithms using asymptotic notations. 
3. Implement linear and non-linear data structures. 
4. Illustrate various techniques for searching and sorting. 
5. Design algorithms using fundamental design techniques: greedy, divide and conquer, and 
dynamic programming.
