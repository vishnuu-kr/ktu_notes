Course Objectives: 
1. Model the systems using transfer function approach as well as state space approach 
2. Employ time domain analysis to predict and diagnose transient performance parameters of the 
system for standard input functions. 
3. Compute the time domain and frequency domain specifications of a system 
4. Analyse dynamic systems for their stability and performance using root locus and  Bode plot 
5. Analyse systems using state space approach
