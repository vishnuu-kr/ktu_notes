Course Objectives: 
1. To design and optimize single-purpose processors and explore ASICs and ASIPs. 
2. To introduce FPGA and behaviour synthesis using VHDL. 
3. To implement controller designs using Arduino. 
4. To address power-aware system design and code efficiency. 
5. To explore DSP applications and address generation units.
