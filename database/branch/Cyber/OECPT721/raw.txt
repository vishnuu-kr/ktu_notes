Course Objectives: 
1. To be able to develop simple programs for 8051 using assembly language programming 
2. To interface 8051 microcontroller with peripheral devices.    
3. To understand the architecture and design concept of embedded systems 
4. To be able to design embedded systems based on Arduino.
