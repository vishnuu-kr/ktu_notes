Course Objectives: 
1. Implement R programs to extract data from different sources, as well as to clean 
and manipulate it. 
2. Develop advanced R programs for data analytics. 
3. Develop advanced Python programs for machine and deep learning. 
4. Analyse the ROS framework, architecture and interface.
