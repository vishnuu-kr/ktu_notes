Course Objectives: 
1. To provide an overview of the nature of human visual perception, the elements of an 
image-processing system which includes sampling, quantisation and two dimensional 
transforms. 
2. To make the students familiar with image enhancement, restoration, segmentation, 
classification, machine learning and deep learning techniques for image analysis.

Suggestion on Project Implementation 
● In the first class, students shall be given an awareness and/ or guidelines on how to implement 
and learn this course through a project and are asked to form groups of maximum five 
members in the first week. 
● Each group should identify a project where different image processing techniques can be used 
for biomedical image processing applications and perform a literature survey. 
● Choose a suitable dataset, proper processing techniques and suitable machine learning, deep 
learning or computer vision algorithms for your application. 
● Develop the block diagram, the algorithms, the flowchart and necessary codes and present 
them before the jury within the first month after the commencement of classes. 
● Implement the necessary algorithms. 
● Prepare a report and present before the jury for final evaluation.
