Course Objectives: 
1. To understand the basic networking models and the OSI layer functionalities 
2. To understand the different protocols used in computer networks

Hours 
1 
Introduction to computer networks: Transmission modes - serial and 
parallel transmission, asynchronous, synchronous, simplex, half duplex, full 
duplex communication. Switching: circuit switching and packet switching  
Networks: Network criteria, physical structures, network models, categories 
of networks, Network models: OSI model, Layers in OSI model, TCP/IP 
protocol suite.  
Physical Layer: Guided and unguided transmission media (Co-axial cable, 
UTP, STP, Fiber optic cable) 
7 
2 
Data Link Layer: Framing, Flow control (stop and wait, sliding window 
flow control) Error control, Error detection (check sum, CRC), Bit stuffing, 
HDLC   
Media access control: Ethernet (802.3), CSMA/CD, Logical link control, 
Wireless LAN (802.11), CSMA/CA 
Network Layer: IPv4 & IPV6, Address Resolution protocols (ARP, RARP) 
Subnetting, Classless Routing (CIDR), ICMP, IGMP, DHCP. 
12 
3 
Networking devices: Hubs, Bridges & Switches 
Routing: Routing and Forwarding, Static routing and Dynamic routing   
9 

 
 
Routing Algorithms:  Distance vector routing algorithm, Link state routing 
(Dijkstra’s algorithm) Routing Protocols: Routing Information protocol 
(RIP), Open Shortest Path First (OSPF), Border Gateway Protocol (BGP), 
MPLS 
4 
Introduction to information system security, common attacks 
Security at Application Layer (E-MAIL, PGP and S/MIME). Security at 
Transport Layer (SSL and TLS). Security at Network Layer (IPSec). 
Defence and counter measures: Firewalls and their types. Limitations of 
firewalls, Intrusion Detection Systems -Host based, Network based, and 
Hybrid Intrusion detection systems
