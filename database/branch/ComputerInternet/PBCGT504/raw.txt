Course Objectives: 
1. To understand the software product development phases. 
2. To develop a software product following a software process model.

Introduction to Software Engineering and Process Models: Software 
engineering, Software characteristics and types, Layers of Software 
Engineering-Process, Methods, Tools and Quality focus.  Software Process

Diagrams in UML(Use  case diagram, Sequence Diagram, Activity diagram)
