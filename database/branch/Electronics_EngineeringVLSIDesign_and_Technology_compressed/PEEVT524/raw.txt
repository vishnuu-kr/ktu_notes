Course Objectives: 
1. Understand the different stages of design flow; the basic data structures and algorithms used 
in each stage 
2. Understand data structures and algorithms used in recent CAD tools 
3. Identify suitable data structures and propose new algorithms for CAD applications and 
develop new CAD tools
