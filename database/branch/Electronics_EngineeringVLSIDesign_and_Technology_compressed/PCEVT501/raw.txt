Course Objectives: 
1. This Course aims to offer distinctive coverage of the dynamic and static circuits and to 
discuss important aspects of timing and synchronisation, power dissipation, interconnect 
packing, and signal integrity in real-world system design

Static and Dynamic circuits: 
CMOS Inverter: Static CMOS Inverter: DC Characteristics, Beta Ratio 
Effects, Noise margin. Other static CMOS logic gates, static properties (2 
input NAND, NOR), Combinational logic circuits.

performance dynamic circuits-Domino CMOS, Multi Output Domino Logic, 
Dual-rail Domino Logic, NP Domino logic (NORA) logic.
