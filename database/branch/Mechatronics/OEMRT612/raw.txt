Course Objectives: 
1. To study about PLC and the different programming languages 
 
2. To understand the various PLC instructions 
3. To understand the need of computer control in automation 
4. To study the data acquisition systems.

PLC PROGRAMMING METHODS 
Architecture, Study of different types of pneumatic and hydraulic valves, 
PLC Programming languages - Relay Logic-Ladder Logic-Programming of 
Gates, PLC Timers-ON delay, OFF delay, Retentive Timers, Programming 
Counters-up/down counter-Combining counter, PLC Arithmetic functions

PLC DATA HANDLING FUNCTIONS 
PLC Program control instructions: Master Control Reset - Skip -Jump & 
Move Instruction, Sequencer instructions, Types of PLC analog modules and 
systems, PLC analog signal processing - Case study of Tank level control 
system, bottle filling system and sequential switching of motors

3 
COMPUTER CONTROL 
Need of computer in a control system-Functional block diagram of a 
computer control system-Data Loggers - Supervisory computer control-

Transducer. Interfacing ADC and DAC with microprocessor / Multiplexer-
Microprocessor/PC based acquisition systems
