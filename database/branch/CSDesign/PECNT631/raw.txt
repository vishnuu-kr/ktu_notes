Course Objectives: 
1. To learn how to use user interface software and technologies in designing human-computer 
interaction, user experience, and ultimately, how we navigate and interact with the digital 
world. 
2. To delve into the principles, methodologies, and advancements that drive the design, 
development, and implementation of user interfaces. 
NB- Students are not expected to write Bootstrap code for the exam.
