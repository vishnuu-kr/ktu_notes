Course Objectives: 
1. The purpose of this course is to make awareness about strong theoretical relationships 
between computer graphics and image processing.  
2. This course helps the learner to understand three dimensional environment representation in a 
computer, transformation of 2D/3D objects, basic mathematical techniques and algorithms 
used to build useful applications, imaging, and image processing techniques.  
3. The study of computer graphics and image processing develops the ability to create image 
processing frameworks for different domains and develops algorithms for emerging display 
technologies.
