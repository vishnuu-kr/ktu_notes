Course Objectives: 
1. To offer learners an introduction to Industry 4.0 and its applications. 
2. To impart knowledge about various data monitoring and control techniques. 
3. To give an overview of the Interconnection and Integration of the Physical World with Cyber 
Space.

Evaluation Methods: 
 
Each student should develop an IoT based system  
 
The system developed should include IoT Protocols and web/smartphone based user 
interface/dashboard for controlling/monitoring the IoT based system 
 
The project details and the outcomes to be evaluated are as follows 
 
Project: Design, simulate and realize an IoT/IIoT based system 
 
 
 
Design Specifications  
 
Detailed specifications of the system are to be made after conducting study of the 
place/facility where the system is to be deployed 

 
 
 
System design 
 
Develop a block diagram of the system 
 
Select sensors, actuators, embedded boards and power source required for the system 
 
Select the communication protocol required for the system 
 
System development 
 
Develop the code for the embedded components of the system 
 
Develop the application code for IoT/IIoT protocols (eg, MQTT, CoAP) 
 
Develop network communication FoG, Edge Gateway etc.. 
 
Develop the user interface/dashboard for controlling/monitoring the IoT based system 
 
Hardware implementation and integration of the system 
 
Assemble the various hardware components of the system 
 
Test the functionality of the system in a controlled environment 
 
Deployment of the system in the field  
 
Sensor data collection and analysis
