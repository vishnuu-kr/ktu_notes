Course Objectives: 
1. To familiarize the basics of Instrumentation system and its quality parameters. 
2. To understand and apply the concepts of various transducers and other measuring instruments.

Suggestion on Project Topics 
Note: 
1.The project proposal submitted by each group (4 to 8 students) must include a measurement system, 
which can be either analog or digital. 
2.The sensor calibration is mandatory. 
3. Students can incorporate suitable signal conditioning circuits, communication standards or      
display   devices in the project work. They can evaluate the outputs and run necessary analytics on the 
data generated. 
4.Sample projects: 
a. Implementation of a temperature transducer for low temperature measurement  
 
b.  Wind measurement system using anemometers  
           c.  Vibration measurement using accelerometers
